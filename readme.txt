Aplicação Bancária

Autor: Everton da Silva Moura

Java 1.8
Servidor WildFly 12
Maven 3.3.9

Bom para iniciar a task eu abri mão do uso de banco, mesmo em memória, pois tomaria mais tempo para escrever a solução,
para realizar alguns testes escrevi uma classe de mock para auxiliar nos cenários; com isso o foco ficou na construção o das 
logicas e validações necessárias para transferências entre contas bem como na divisão de ações para cada responsável no processo;

Muitas novidades podem ser adicionados em uma versão futura como por exemplo, log, controle de autenticação da conta, mais operações além de transferência bancaria, consultas dos lançamentos diários.
